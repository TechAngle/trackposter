# Contributing to Trackposter
## To start with...
*Thanks for help for this project! I highly appreciate it.*

> If it's possible, please share this repository wherever you can, it helps the project become more useful for people.

## How to contribute
### Report bug/propose an enhancement
> **IMPORTANT**: Please, check if bug is not on your side. Also, you must agree that all content is completely authored by you.

If you want to report a bug, you can create an issue on [Issues](https://github.com/TechAngle/trackposter/issues) page.
- Open an Issue.
- Provide as much context as you can about what youâ€™re running into.
- Provide project and platform versions (go version, OS, etc), depending on what seems relevant.

If you want to **suggest an enhancement** (add new features or suggest updates of existing ones) - you can open an issue:
- Make sure that enhancement was not suggested before and you use the latest version.
- Open an Issue and provide clear and descriptive title
- Write a step-to-step description of your enhancement and its aim, explain why you think it would be useful for the project.
